Introduction

This is a demo tutorial for the Designing the Data Science Classroom workshop at rstudio::conf 2022.

Packages

We will use the tidyverse package in this tutorial. Run the code below to load the package.

library(tidyverse)

Data: Ikea furniture

The data set is all about Ikea furniture. It was obtained from the TidyTuesday data collection.

Run the code below to see the data.

ikea

We will primarily use the following variables in the analysis:

  • category: The furniture category that the item belongs to (Sofas, beds, chairs, Trolleys,…)
  • price_usd: Price in US dollars based on the price on the Ikea website on 4/20/2020.

Run the code to see the names of all variables in the data set, and click here to see the full data dictionary.

names(ikea)
